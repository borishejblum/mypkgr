# Copy of mvtnorm::dmvnorm for progiling and reference

Copy of mvtnorm::dmvnorm for progiling and reference

## Usage

``` r
my_dmvnorm(
  x,
  mean = rep(0, p),
  sigma = diag(p),
  log = FALSE,
  checkSymmetry = TRUE
)
```

## Arguments

- x:

  matrix

- mean:

  vector

- sigma:

  covariance matrix

- log:

  logical. Default is `FALSE`

- checkSymmetry:

  logical. Default is `TRUE`
